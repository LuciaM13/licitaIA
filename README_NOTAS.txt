
# Notas de esta versión

Esta versión usa **solo** los datos que aparecen en:
- el CSV / Excel de precios (`datos.csv`)
- el documento de presupuesto / pliego

No se usan:
- factores de reurbanización inventados
- porcentajes de seguridad y salud o gestión ambiental no dados como fórmula
- colchón de seguridad del 10 %

Los únicos porcentajes aplicados como fórmula son:
- 13 % Gastos Generales
- 6 % Beneficio Industrial
- 21 % IVA
